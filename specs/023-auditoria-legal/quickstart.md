# Quickstart: AB-38 — Auditoría y Cumplimiento Legal

**Prerequisites**
- Service running on port 8080
- JWT token: `$ADMIN_TOKEN` (role: ADMIN, user: admin@accioneselbosque.com)
- JWT token: `$INVESTOR_TOKEN` (role: INVESTOR, user: investor@test.com)
- Audit log pre-populated with events from multiple investors (generated by login attempts, order placements, etc.)

---

## Flow 1: Query the full audit log (paginated, default filters)

```bash
curl -s -X GET "http://localhost:8080/audit/events?page=0&size=50" \
  -H "Authorization: Bearer $ADMIN_TOKEN" | jq .
```

Expected 200:
```json
{
  "content": [
    {
      "id": 10001,
      "eventType": "AUTH_FAILURE",
      "investorId": 42,
      "performedBy": 42,
      "referenceType": null,
      "referenceId": null,
      "detail": { "reason": "INVALID_PASSWORD", "attempt": 3 },
      "result": "FAILURE",
      "ipAddress": "192.168.1.100",
      "archived": false,
      "occurredAt": "2026-05-13T10:15:00Z"
    }
  ],
  "totalElements": 5420,
  "page": 0,
  "size": 50
}
```

DB verification:
```sql
SELECT COUNT(*) FROM audit_events WHERE archived = false;
```

## Flow 2: Filter audit log by investor ID and date range

```bash
curl -s -X GET "http://localhost:8080/audit/events?investorId=42&from=2026-05-01&to=2026-05-14&page=0&size=20" \
  -H "Authorization: Bearer $ADMIN_TOKEN" | jq .
```

Expected 200:
```json
{
  "content": [
    {
      "id": 10005,
      "eventType": "ORDER_CREATED",
      "investorId": 42,
      "performedBy": 42,
      "referenceType": "ORDER",
      "referenceId": 7801,
      "detail": { "symbol": "ECOPETROL", "quantity": 100, "price": 1972.00, "orderType": "MARKET" },
      "result": "SUCCESS",
      "ipAddress": "192.168.1.100",
      "archived": false,
      "occurredAt": "2026-05-12T14:23:00Z"
    },
    {
      "id": 10004,
      "eventType": "AUTH_SUCCESS",
      "investorId": 42,
      "performedBy": 42,
      "referenceType": null,
      "referenceId": null,
      "detail": { "device": "Chrome/Windows" },
      "result": "SUCCESS",
      "ipAddress": "192.168.1.100",
      "archived": false,
      "occurredAt": "2026-05-12T14:20:00Z"
    }
  ],
  "totalElements": 14,
  "page": 0,
  "size": 20

```

DB verification:
```sql
SELECT id, event_type, result, occurred_at
FROM audit_events
WHERE investor_id = 42
  AND occurred_at BETWEEN '2026-05-01' AND '2026-05-14 23:59:59'
ORDER BY occurred_at DESC;
```

## Flow 5: Export audit log as CSV

```bash
curl -s -X GET "http://localhost:8080/audit/events/export?eventType=ORDER_CREATED&from=2026-05-01&to=2026-05-14" \
  -H "Authorization: Bearer $ADMIN_TOKEN" \
  -o audit-log.csv \
  -w "%{http_code} | Content-Type: %{content_type}"
```

Expected 200 with headers:
```
Content-Type: text/csv
Content-Disposition: attachment; filename="audit-log.csv"
```

Expected CSV preview (first lines):
```
id,eventType,investorId,performedBy,referenceType,referenceId,result,ipAddress,archived,occurredAt
10005,ORDER_CREATED,42,42,ORDER,7801,SUCCESS,192.168.1.100,false,2026-05-12T14:23:00Z
```

## Flow 6: Reject access from a non-ADMIN role

```bash
curl -s -X GET "http://localhost:8080/audit/events?page=0&size=20" \
  -H "Authorization: Bearer $INVESTOR_TOKEN" | jq .
```

Expected 403:
```json
{
  "error": "FORBIDDEN",
  "message": "Acceso denegado. Se requiere rol ADMIN."
}
```
